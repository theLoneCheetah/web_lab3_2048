# 2048

[Деплой на GitHub Pages](https://thelonecheetah.github.io/web_lab3_2048/)

## Задача
* Создать веб-версию игры 2048, где реализовать: взаимодействие с игровым полем, запись и просмотр таблицы лидеров, возврат шага назад, продолжение сессии с последнего игрового действия, рестарт игры.
* Не использовать CSS-фреймворки (Bootstrap, Foundation и т.д.) и JS-фреймворков (Angular, React, Vue и т.д.).
* Не использовать инструменты innerHTML/outerHTML.
* Предоставить deploy проекта.

## Реализованные функции
* Создана базовая HTML-страница: контейнер для тайлов, блок с текущим количеством очков, модальное окно с лидербордом и предложением начать игру заново.
* Сохранение результата под вводимым именем в таблице лидеров при окончании игры.
* Таблица рекордов отражает в каждой строке имя пользователя, количество очков и дату.
* Дизайн адаптирован под десктопные (управление стрелочками на клавиатуре) и мобильные устройства (управление с помощью свайпов).
* Игра завершается автоматически при достижении плитки 2048 или невозможности сделать ход.
* Логика игры позволяет выполнять несколько слияний даже в одной строке, например, [4, 4, 4, 4] -> [16, 0, 0, 0] при движении влево.
* Текущий счёт пользователя обновляется в соответствии с суммой чисел, которые были указаны на плитках перед слиянием.
* Новые плитки (1-2) со значениями 2 или 4 добавляются случайно на свободных клетках сетки.
* Каждая плитка имеет свой цвет.
* Данные об игровом поле, истории и таблице лидеров сохраняются в local storage и доступны при перезагрузке страницы.
* Реализована плавная анимация появления, удаления, перемещения и слияния плиток.
